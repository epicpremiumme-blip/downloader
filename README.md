## Downloaded Files

Check the [downloads folder](./downloads) for downloaded files.

### How to extract split files:
1. Download all parts (.zip, .z01, .z02...)
2. Open the main .zip file with WinRAR or 7-Zip
3. Extract normally - parts combine automatically
